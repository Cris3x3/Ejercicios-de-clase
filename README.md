# Estructura del Proyecto

Este proyecto está organizado de la siguiente manera:

## Directorios Principales

- **src/**: Contiene el código fuente del proyecto.
  - **componentes/**: Reutilizables componentes utilizados en el proyecto.
  - **páginas/**: Diferentes secciones o páginas de la aplicación.

- **ejercicios/**: Este directorio alberga ejercicios prácticos relacionados con las clases.

- **tests/**: Aquí se encuentran las pruebas automatizadas para asegurar que el código funcione correctamente.

## Organización de Ejercicios

Cada ejercicio está organizado en su propio subdirectorio dentro del directorio `ejercicios/`, donde cada subdirectorio sigue la estructura de:
- **ejercicio-n/**: Contiene todos los archivos necesarios para el ejercicio.
- **README.md**: Un archivo que explica el ejercicio, su propósito y cómo configurarlo.

## Instrucciones de Uso

Para ejecutar los ejercicios, navega al directorio correspondiente y sigue las instrucciones en el `README.md`.

---

Espero que esta estructura ayude a mantener el proyecto organizado y claro para todos los colaboradores.
